Career Journey

The throughline across eleven years is this: every role built directly on the last one. The reconciliation algorithms written in Java 7 at Tekizma India sharpened the instinct for data consistency that later drove the Kafka/CDC architecture at the oncology platform. The NLP research and Drools rule engine work were precursors to the GenAI agents shipped a decade later. The batch-processing frameworks built for the State Health Insurance Marketplace — handling large-scale eligibility renewals — informed the multi-threaded thinking behind Databricks clinical pipelines.

The career arc is: individual contributor → product architect → consulting lead → engineering manager — each transition deliberate, each domain deepening a core specialisation in healthcare systems at scale.
